## v1.0.0（2026-07-20）

**Kairos 记忆系统架构草稿完善阶段发布。**

### 核心能力

- 四轴正交度量空间（使用价值/见证价值/时间/认知完整性；完整认知模型含第五轴可及性轴，v1.1 实现）+ 潜伏势能
- 六项核心机制（编码/巩固/遗忘/更新/检索/前瞻保持）
- 多重记忆（情景/叙事/语义/程序）+ 四种关系（因果/独立/层级/竞争）
- P1–P6 设计原则 + 辞典式排序七级链 + 价值独立性公理
- 分域真理观（实用论/融贯论/符合论）+ 他律性降级契约
- 系统上限三维（校准锚定/承载/反身性）+ 弱自反性前提

### 架构

- 六层架构 + 监督平面：宪法主权面 / 元认知层 / 策略层 / 存储层 / WM 层 / 接入层 + 监督平面
- 见证锚定（主副本-强一致性）与使用权重（影子副本-最终一致性）分离
- 四类使用契约（常驻/按需/环境/临时）——连续激活权重的离散投影
- 路径空间 `kairos://` 为第一检索手段，向量搜索退居辅助
- 推理皮层常设 WM 层（最小必要回路：前瞻监控/事件排序/候选裁剪）
- 三级架构梯度（内核级含简化宪法主权面保留外部校准端口）
- 跨层三环不变量（再巩固环/漂移检测环/外部校准环）
- 证伪响应机制（核心命题证伪/轴耦合证伪/VAD 独立性证伪）
- 19 条安全红线（S-01~S-19）+ 链式 HMAC 审计

### 关键变更（v1.0.0 设计阶段）

- 审计庭从宪法主权面剥离至独立监督平面（§1.7）
- 内核级梯度加入简化宪法主权面（保留外部校准端口）
- 差异检验分级触发 + 良性正相关标记
- 耦合计监测器 + VAD 独立性测试器 + 体系聚合证伪审计器
- 自我参照效应偏置监测 + 偏置放大率监测 + 认知关节累积熵监测
- 监督平面降级协议 + 跨层协议绝对上限（10 条）
- 表征降级门控（编码门控的反向操作）
- 真理模式切换协议增加前瞻触发条件
- 内隐偏向归属声明（横切属性，不归属单一使用类型）
