# Kairos 非功能需求规格

> **定位**：将架构 §8 中散落的量化指标收拢为一份可验证的需求表。架构文档约束「正确性」，本文约束「性能」。

---

## 一、性能

| 指标 | P50 | P95 | P99 | 测量方法 |
|:----|:---:|:---:|:---:|:---------|
| 记忆写入延迟（单条） | ≤ 50ms | ≤ 200ms | ≤ 500ms | 写入计时 |
| 记忆检索延迟（路径前缀） | ≤ 20ms | ≤ 80ms | ≤ 200ms | 检索计时（10 万条存储） |
| 记忆检索延迟（语义相似度） | ≤ 100ms | ≤ 300ms | ≤ 800ms | 检索计时（10 万条存储） |
| 多路径融合延迟 | ≤ 50ms | ≤ 150ms | ≤ 400ms | 融合计时 |
| 写入吞吐 | ≥ 100 ops/s | — | — | 持续写入压测 |
| 检索吞吐 | ≥ 500 ops/s | — | — | 并发检索压测 |
| 升华批处理（单条 raw→behavior） | ≤ 30s | ≤ 120s | — | 全阶段计时 |
| 遗忘扫描（10 万条） | ≤ 60s | — | — | 全表扫描计时 |

## 二、容量

| 指标 | 标准模式 | 轻量模式 | 说明 |
|:----|:--------:|:--------:|:-----|
| 最大记忆数 | ≥ 100 万条 | ≥ 5 万条 | 含已归档 |
| 单条内容上限 | 64 KB | 64 KB | —
| 路径空间深度 | ≤ 10 层 | ≤ 10 层 | —
| 关系索引数 | ≥ 200 万条 | ≥ 5 万条 | —
| 向量维度 | 1536（text-embedding-3-small） | 1024（BGE-M3） | —
| 事件总线缓冲 | ≥ 128 条 | ≥ 64 条 | 背压前 |

## 三、可用性

| 指标 | 目标 | 说明 |
|:----|:----:|:-----|
| 系统可用性 | ≥ 99.9% | 计划内维护除外 |
| 启动时间 | ≤ 10s | 标准模式 |
| 故障恢复 | ≤ 30s | 自动恢复，不丢已持久化数据 |
| 降级可用性 | ≥ 99% | 外部校准中断期间 |

## 四、资源

| 指标 | 标准模式 | 轻量模式 |
|:----|:--------:|:--------:|
| 内存（空闲） | ≤ 256 MB | ≤ 64 MB |
| 内存（全负载） | ≤ 1 GB | ≤ 256 MB |
| CPU（空闲） | ≤ 5% | ≤ 2% |
| 磁盘（10 万条） | ≤ 2 GB | ≤ 500 MB |
| 磁盘年增长（10 万条/月） | ≤ 2 GB | — |

## 五、安全

| 指标 | 要求 |
|:----|:-----|
| 安全红线覆盖率 | 100%（16/16） |
| 审计链完整性 | HMAC-SHA256 可验证 |
| 敏感信息脱敏 | 自动打标 + 可逆脱敏导出 |
| API Key 轮换 | 支持热轮换，无需重启 |
| LLM 日预算 | 可配置，超限熔断 |
