## 理念约束关系

### 层级约束（上级约束下级）

| 上级 | 下级 | 约束方式 |
|:----|:----|:---------|
| 他律性约束 | 宪法主权面 | 外部校准源决定宪法级偏好的注入上限 |
| 宪法主权面 | 价值独立性公理 | S-14 语境自指禁令不可被任何内部决策覆盖 |
| 价值独立性公理 | 辞典式排序 | 排序链中"身份>探索>宪法>校准>认知完整性>时间>间接度"为七级链（详见架构 §3.2 辞典式裁决器），此处"身份>校准>时间"为简化表达，突出见证优先于使用的核心理念 |
| 价值独立性公理 | P1-P4 | P1 承认使用价值但不允许使用改写见证 |
| 辞典式排序 | 分域真理观 | 跨域冲突时辞典式排序兜底 |
| 分域真理观 | 保守倾向 | 跨域回退规则由分域真理观的规范真理定位决定 |
| 保守倾向 | P5 探索 | 保守倾向与 P5 为正交决策变量（认知基础 E.5）：保守倾向决定「方向」（平局时 NO-OP），P5 决定「幅度」（探索预算分配）；两者在平局场景下协作用于控制探索扩张 |
| P6 | 全链 | 任何标量化操作须遵守 P6 受控例外 |

### 并行协作（同级理念间的交互）

| 理念 A | 理念 B | 协作方式 |
|:-------|:-------|:---------|
| P1（使用定义价值） | P3（激活-存储解耦） | P1 提供使用价值度量，P3 确保度量不影响存储结构 |
| 辞典式排序 | 保守倾向 | 辞典式排解决定价值优先，保守倾向处理不可判定场景 |
| 分域真理观 | 价值独立性公理 | 分域真理观确定不同域的真理标准，价值独立性公理确保使用与见证不混淆 |
| P5（探索） | P6（禁止聚合） | P5 的探索产物转化门槛受 P6 约束——转化评估不可无声丢失维度信息 |

### 张力对（需要显式管理的冲突）

| 冲突双方 | 张力性质 | 管理机制 |
|:---------|:---------|:---------|
| P1（使用定义价值） vs 价值独立性公理 | 使用价值主导操作 vs 见证价值不可被使用覆盖 | 分域真理观：操作层实用论主导，冲突层符合论兜底 |
| P5（探索） vs 保守倾向 | 探索扩展边界 vs 保守不引入风险 | 探索安全边界（P5 探索安全边界（认知基础 E.5）：不挤占核心记忆） + 保守偏向告警 |
| 他律性约束 vs 突破上限目标 | 外部依赖 vs 自主演化 | 弱自反性声明 + 三阶段演进（纯他律→受限自主→混合自主） |
| 构造主体立场 vs 他律性 | 身份自我生成 vs 外部校准定义身份 | 当前诚实标注为开放问题，留待受限自主阶段解决 |

## 理念到架构的映射链

```
认知基础理念 → 架构约束 → 架构组件 → 验证机制

例 1: 价值独立性公理 → S-14 语境自指禁令 → 见证锚定/使用权重双副本分离 → §5 差异检验
例 2: 辞典式排序 → 七级链为宪法级约束 → §3.2 辞典式裁决器 → §3.2 序数压制幅度记录 + 排列漂移审计
例 3: 分域真理观 → 三域三真理切换规则 → §3.2 真理模式切换协议 → 切换条件判定 + 回退兜底
例 4: P6 → 禁止无声丢失维度信息 → 受控例外标注 + 多维表征保留 → E.7 统一审计框架
```
